Basics of VIM

insert mode:
      i - insert mode (type what you want)
    esc - leave insert mode, default is command mode

command mode:
    esc - enter command mode    
     :q - quit from command mode
     :w - write (save)
    :wq - write and quit
      u - undo last action
 ctrl+r - redo
     dd - delete line
[num]dd - delete # of lines

find string stuff:
/string - finds match string "n" to go to next item "N" go previous, "i" to begin editing from there
:%s/string/replacer/("g" to do all)("c" to confirm each) - find and replace
